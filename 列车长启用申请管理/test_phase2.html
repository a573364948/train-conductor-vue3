列车长管理系统 - 第二阶段测试

测试说明

请打开系统主页面（index.html），按照以下步骤进行测试：

1. 部门选择器测试

  • 打开"系统设置"页面，添加一个新部门（如：测试部门）
  • 然后检查以下位置的部门选择器是否包含新部门：
    • 列车长信息管理 - 添加列车长
    • 列车长信息管理 - 编辑列车长
    • 申请管理 - 新建申请
    • 列车长信息管理 - 部门筛选
    • 申请管理 - 部门筛选
    • 统计与报表 - 部门筛选

2. 数据一致性测试

  • 在"统计与报表"页面，点击"编辑定员"
  • 修改某个部门的定员数量
  • 检查部门统计表格是否正确更新
  • 刷新页面，检查数据是否保持一致

3. UI/UX测试

  • 打开任意模态框（如添加列车长）
  • 填写部分信息但不提交
  • 关闭模态框
  • 重新打开同一模态框，检查表单是否已清空

预期结果

✓ 所有部门选择器都应该显示新添加的部门
✓ 部门定员修改后，统计数据应该立即更新
✓ 模态框关闭后重新打开，表单应该是空的
